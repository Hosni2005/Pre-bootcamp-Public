# Song Track List

## Description

This project demonstrates the use of Java's HashMap collection. Songs are stored using their track titles as keys and sample lyrics as values. The application allows lyrics to be retrieved instantly by song title and displays all stored tracks with their corresponding lyrics.

## Features

* Create a HashMap called `trackList`
* Store song titles as keys
* Store lyrics as values
* Retrieve a song's lyrics using its title
* Display all tracks and lyrics

## Technologies Used

* Java
* HashMap
* Loops
* Collections Framework
