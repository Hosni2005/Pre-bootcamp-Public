# Java Cafe Orders

## Description

This project demonstrates Object-Oriented Programming concepts in Java by creating custom classes for managing cafe orders.

The project uses an `Item` class to store product information and an `Order` class to manage customers, items, totals, and order status.

## Features

* Create custom data types using classes
* Use constructors
* Use overloaded constructors
* Apply private access modifiers
* Implement getters and setters
* Add items to orders
* Calculate order totals
* Track order readiness
* Display order information

## Classes

### Item Class

Stores:

* Item name
* Item price

Includes:

* Constructor
* Getters
* Setters

### Order Class

Stores:

* Customer name
* Ready status
* List of items

Includes:

* Constructors
* addItem()
* getStatusMessage()
* getOrderTotal()
* display()
