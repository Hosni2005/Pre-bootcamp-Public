Yelp Database Design Assignment
Project Overview

This assignment focuses on designing a Yelp-style database system using an Entity Relationship Diagram (ERD) and database modeling concepts. The system stores information about restaurants, food items, users, and reviews.

Features
Store restaurant details
Manage food/menu items
Store user accounts
Allow users to add reviews and ratings
Create relationships between tables using Primary Keys and Foreign Keys
