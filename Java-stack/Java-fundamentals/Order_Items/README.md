# Java Cafe App - Object Oriented Programming

## Description

This project is a Java Cafe application that demonstrates the basics of Object-Oriented Programming (OOP) by creating custom classes to organize cafe data.

The application separates data into different classes, making the code easier to manage and maintain.

## Features

* Create custom classes using Java
* Store item information using an `Item` class
* Store order information using an `Order` class
* Create objects using the `new` keyword
* Manage customer orders
* Track order totals
* Update order status
* Store multiple items inside orders using `ArrayList`

## Classes

### Item Class

Stores cafe menu item information:

* Item name
* Item price

Example items:

* Mocha
* Latte
* Drip Coffee
* Cappuccino

### Order Class

Stores order details:

* Customer name
* Total price
* Ready status
* List of ordered items

Compile:

```bash

```

Run:

```bash

```
