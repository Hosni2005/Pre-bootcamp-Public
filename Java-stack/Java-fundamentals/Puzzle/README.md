# Puzzle Java

## Description

Puzzle Java is a Java Fundamentals project that demonstrates the use of random number generation, ArrayLists, loops, strings, and methods. The application generates random numbers, random letters, passwords, and collections of passwords while providing practice with Java utility classes.

## Features

### getTenRolls()

Generates and returns an ArrayList containing 10 random integers between 1 and 20.

### getRandomLetter()

Returns a random lowercase letter from the English alphabet.

### generatePassword()

Creates and returns a random password consisting of 8 randomly generated letters.

### getNewPasswordSet(int length)

Generates and returns a collection of random passwords. The number of passwords generated is determined by the provided length parameter.

### shuffleArray() (Sensei Bonus)

Randomly shuffles the elements of an ArrayList.
