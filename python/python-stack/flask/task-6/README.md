# 🍓 Dojo Fruit Store

A simple Flask web application that allows users to order fruits online and view their checkout summary.  
Built using **Python Flask**, **HTML**, **Bootstrap**, and **Sessions**.

---

## 🚀 Features

- Select fruit quantities
- Submit customer information
- Checkout page with order summary
- Session handling using Flask
- Separate fruits showcase page
- Bootstrap responsive design

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- Bootstrap 4
- Jinja2 Templates

## ▶️ How to Run

1. Clone the repository

```bash
git clone <your-repo-link>
```

2. Navigate to the project folder

```bash
cd dojo-fruit-store
```

3. Install Flask

```bash
pip install flask
```

4. Run the server

```bash
python server.py
```

5. Open in browser

```bash
http://127.0.0.1:5000
```
