# List Functions Assignment

This project contains multiple Python functions that practice working with lists, loops, conditions, calculations, and data analysis.

---

## Functions Included

### 1. `biggie_size()`

Replaces all positive numbers in a list with the string `"big"`.

#### Example

```python
biggie_size([-1, 3, 5, -5])
```

#### Output

```python
[-1, "big", "big", -5]
```

---

### 2. `count_positive()`

Counts how many positive numbers exist in a list and replaces the last value with the count.

#### Example

```python
count_positive([-1, 1, 1, 1])
```

#### Output

```python
[-1, 1, 1, 3]
```

---

### 3. `sum_total()`

Returns the sum of all values in a list.

#### Example

```python
sum_total([1, 2, 3, 4])
```

#### Output

```python
10
```

---

### 4. `average()`

Calculates the average of all numbers in a list.

#### Example

```python
average([1, 2, 3, 4])
```

#### Output

```python
2.5
```

---

### 5. `length()`

Returns the total number of elements in a list.

#### Example

```python
length([1, 2, 3, 4, 5, 8])
```

#### Output

```python
6
```

---

### 6. `minimum()`

Finds and returns the smallest value in a list.

#### Example

```python
minimum([37, 2, 1, -9])
```

#### Output

```python
-9
```

---

### 7. `maximum()`

Finds and returns the largest value in a list.

#### Example

```python
maximum([37, 2, 1, -9])
```

#### Output

```python
37
```

---

### 8. `ultimate_analysis()`

Returns a dictionary containing:

* Total sum
* Average
* Minimum value
* Maximum value

#### Example

```python
ultimate_analysis([37, 2, 1, -9])
```

#### Output

```python
{
    'sumTotal': 31,
    'average': 7.75,
    'minimum': -9,
    'maximum': 37
}
```
