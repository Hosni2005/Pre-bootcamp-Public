📌 Python Functions Practice

This project contains a set of simple Python functions designed to practice fundamental programming concepts such as loops, lists, conditions, and function behavior.

📖 Overview

The code includes multiple utility functions that perform different operations on numbers and lists. Each function focuses on a specific concept to strengthen problem-solving skills.

⚙️ Functions Included
1. Countdown

Creates a list counting down from a given number to 0.

countdown(5)
# Output: [5, 4, 3, 2, 1, 0]
2. Print and Return

Prints the first element of a list and returns the second element.

print_function([1, 2])
# Output:
# 1
# 2
3. First Plus Length

Returns the sum of the first element in the list and the length of the list.

sum_first_length([1, 2, 3, 4, 5])
# Output: 6
4. Values Greater Than Second

Creates a new list with values greater than the second element in the original list.
Prints the count of such values.

value_greater_than_second([5, 2, 3, 2, 1, 4])
# Output:
# 3
# [5, 3, 4]

If the list has less than 2 elements:

value_greater_than_second([3])
# Output: False
5. Length and Value

Creates a list of a given size where all elements have the same value.

length_and_value(4, 7)
# Output: [7, 7, 7, 7]
🚀 How to Run
Make sure you have Python installed.
Copy the code into a .py file.

python filename.py
