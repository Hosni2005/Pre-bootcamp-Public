# Cafe Java

## Overview

Cafe Java is a beginner Java project that simulates a coffee shop ordering system. The application demonstrates the use of variables, conditional statements, arithmetic operations, and formatted output.

The program manages customer orders, checks whether orders are ready, calculates totals, and displays messages to customers.

---

## Features

### Customer Greetings

Displays a welcome message for customers entering the cafe.

Example:

```text
Welcome to Cafe Java, Shatha
```

### Order Status Notifications

Checks whether an order is ready and displays the appropriate message.

Example:

```text
Sali, your order is ready
```

or

```text
Adam, your order will be ready shortly
```

### Order Total Calculation

Calculates and displays the total cost of customer orders.

Example:

```text
Your total is $6.4
```

### Price Difference Calculation

Calculates the difference between the charged item and the correct item price.

Example:

```text
Your total is $0.5
```
